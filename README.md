# Ecommerce Project

## Prerequisites
- Node.js
- PostgreSQL

## How to Run

1.  **Start the Database**
    Ensure PostgreSQL is running:
    ```bash
    brew services start postgresql@14
    ```

2.  **Install Dependencies** (First time only)
    ```bash
    npm install
    ```

3.  **Start the Server**
    ```bash
    npm run dev
    ```

4.  **Access the Application**
